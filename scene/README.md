# Astro Boilerplate

## Требования

- [Node.js](https://nodejs.org/en/) версии 22+. Проверить версию ноды после установки можно командой **node -v**.
- [pnpm](https://pnpm.io/ru/)

## Как начать работу

1. Перейдите в папку `frontend`, установите npm-зависимости: `pnpm i`.
2. Запустите команду `pnpm run dev`.
3. Перейдите по адресу [http://localhost:3000](http://localhost:3000) в браузере.

## Сборка проекта

1. Перейдите в папку `frontend`, установите npm-зависимости: `pnpm i`.
2. Запустите команду `pnpm run build`. Создастся папка `build`.
